Python中的装饰器

发表于 2020-11-21 更新于 2020-11-22
本文字数： 1.6k 阅读时长 ≈ 1 分钟

装饰器就像你有杯摩卡咖啡，往上面再添一些巧克力碎，这个添加动作就是装饰，巧克力碎就是装饰器。

1 什么时候使用装饰器？

可以在不修改原始代码的情况，为已经存在的对象添加一些额外的功能。常用的需求场景就是：日志功能、性能测试等。Python中的修饰器的设计其实就是设计模式中的装饰者模式。

装饰器本身可以是函数也可以是类，如果使用的是类装饰器，那么使用@形式装饰时，其实上调用的是类装饰器的__call__方法。
